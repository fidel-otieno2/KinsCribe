## Color Specifications

### @Mention Badge
```css
color: #8b5cf6;                      /* Purple text */
background: rgba(139, 92, 246, 0.15); /* Light purple background */
font-weight: 700;                     /* Bold */
padding: 2px 6px;                     /* Padding */
border-radius: 6px;                   /* Rounded corners */
```

Visual: `@john_smith` appears as:
```
┌──────────────┐
│ @john_smith  │  ← Purple text on light purple background
└──────────────┘
```

### #Hashtag Badge
```css
color: #c4a35a;                      /* Gold text */
background: rgba(196, 163, 90, 0.1); /* Light gold background */
font-weight: 600;                     /* Semi-bold */
padding: 2px 6px;                     /* Padding */
border-radius: 6px;                   /* Rounded corners */
```

Visual: `#family` appears as:
```
┌─────────┐
│ #family │  ← Gold text on light gold background
└─────────┘
```


## Summary

✅ **Purple badge** with light background - Distinct from regular text
✅ **Bold text** - Easy to spot
✅ **Rounded corners** - Modern, polished look
✅ **Tappable** - Clear interaction
✅ **Floating dropdown** - Above keyboard, doesn't block content
✅ **User avatars** - Visual recognition
✅ **Real-time filtering** - Instant results
✅ **Notifications** - Push + in-app alerts

The mention system is visually distinct, easy to use, and works exactly like Instagram! 🎉
